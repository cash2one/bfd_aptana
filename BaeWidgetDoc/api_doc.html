原子级组件通用格式

{
"id": "1_legend",
"type": "legend",
"container": "menu",
"cols": [{                 
    "key": "uv",
    "title": "唯一访客数 ",
    "desc": "鼠标滑上去出现的解释",
    "format":"num",
  "groupBy":0 //默认为0，groupBy中元素的索引值
}]，
"groupBy": ["state_date"],
"dataParam":{
"sort": "state_date", 
     "sortMode": "asc"//在请求数据时表示排序方式asc或desc
},
"events": [{  
     "name": "change",
     "target": "1_chart",//被影响对象id
     "action": "flush"//被影响对象响应函数名
 }, {
     "name": "change",//此例中legend的change事件影响2个对象 id=1的chart，id=10的selectbox
    "target": "10_selectbox",
    "action": "flush"
}],
"option": {}//这里存各组件的自定义setting
}]
}
配置项 描述
id 必须项，格式为"数据库id_组件type"，作为组件在页面中的唯一标识
type 必须项，组件的type
container 必须项，放置组件的div的id
cols cols数组中传入数据项的key（必须项）,title（名称，必须项）,desc（描述），format（数据格式类型）,属性可扩展
groupBy groupBy数组中传入用于分组的列的key
dataparam dataparam中为自定义参数，在数据请求中直接将dataparam中所有字段传入后台
sort 在请求数据时要求的用于排序的列的key
sortMode 在请求数据时表示排序方式asc或desc
events 各组件加入events，events数组可存入多个事件，name为事件名称，target为被影响的组件的id，action为被影响组件在事件触发的执行自己的方法的名称。
option 这里存各组件的自定义setting
